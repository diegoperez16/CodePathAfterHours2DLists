2D Array Indexing 🎯

Access elements using: array[row][column]

Both row and column indices start at 0!

mansion = [
    ["👻", "🎃", "🦇"],
    ["🕷️", "🧛", "👻"],
]

print(mansion[0][0])  # Output: 👻
print(mansion[1][1])  # Output: 🧛
print(mansion[0][2])  # Output: 🦇

Traversing with Nested Loops 🔄

To visit every room in the haunted mansion, we need nested loops!

Outer loop → Rows (floors)

Inner loop → Columns (rooms)

mansion = [
    ["👻", "🎃", "🦇"],
    ["🕷️", "🧛", "👻"],
]

# Visit every room!
for row in range(len(mansion)):
    for col in range(len(mansion[row])):
        print(f"[{row}][{col}]: {mansion[row][col]}")

Problem #1: Count the Ghosts 👻

Write a function count_ghosts(mansion) that counts how many ghosts (👻) are in the entire haunted mansion. 

mansion = [
    ["👻", "🎃", "🦇"],
    ["🕷️", "👻", "👻"],
    ["💀", "🦇", "🕸️"]
]

count_ghosts(mansion)
# Output: 3

Solution Breakdown 🧠

1. Define the function that takes a 2D array

2. Initialize a counter = 0

3. Use nested loops to check every room

4. If we find a "👻", increment counter

5. Return the counter

Solution - Step-by-Step 🎯

def count_ghosts(mansion):
    # Initialize counter
    count = 0
    
    # Outer loop: iterate through rows
    for row in range(len(mansion)):
        # Inner loop: iterate through columns
        for col in range(len(mansion[row])):
            # Check if current element is a ghost
            if mansion[row][col] == "👻":
                count += 1
    
    # Return total count
    return count

Problem #2: Trick-or-Treat Total 🍬

Write a function total_candy(neighborhood) that returns the total number of candies collected from all houses. 

neighborhood = [
    [5, 10, 3],
    [7, 2, 8],
    [4, 6, 9]
]

total_candy(neighborhood)
# Output: 54

Solution Breakdown 🧠

1. Define the function

2. Initialize total = 0

3. Use nested loops to visit each house

4. Add each house's candy count to total

5. Return total

Solution - Step-by-Step 🎯

def total_candy(neighborhood):
    # Initialize total
    total = 0
    
    # Loop through each row (street)
    for row in range(len(neighborhood)):
        # Loop through each column (house)
        for col in range(len(neighborhood[row])):
            # Add candy from this house
            total += neighborhood[row][col]
    
    # Return total candy
    return total

Problem #3: Find the Vampire 🧛

Write a function find_vampire(mansion) that returns the position [row, col] of the vampire (🧛). If there's no vampire, return None. 

mansion = [
    ["👻", "🎃", "🦇"],
    ["🕷️", "🧛", "👻"],
    ["💀", "🦇", "🕸️"]
]

find_vampire(mansion)
# Output: [1, 1]

Solution Breakdown 🧠

1. Define the function

2. Use nested loops to search each room

3. When we find "🧛", return [row, col]

4. If we finish searching and didn't find it, return None

Solution - Step-by-Step 🎯

def find_vampire(mansion):
    # Loop through rows
    for row in range(len(mansion)):
        # Loop through columns
        for col in range(len(mansion[row])):
            # Check if we found the vampire
            if mansion[row][col] == "🧛":
                # Return position immediately
                return [row, col]
    
    # If we get here, no vampire found
    return None

Problem #4: Spooky Row Sums 🎃

Write a function row_sums(grid) that prints the sum of each row in a 2D array. 

grid = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
]

row_sums(grid)
# Output:
# Row 0: 6
# Row 1: 15
# Row 2: 24

Solution Breakdown 🧠

1. Define the function

2. Loop through each row

3. For each row, initialize a sum = 0

4. Loop through columns in that row and add to sum

5. Print the row number and its sum

Solution - Step-by-Step 🎯

def row_sums(grid):
    # Loop through each row
    for row in range(len(grid)):
        # Initialize sum for this row
        row_sum = 0
        
        # Loop through each column in this row
        for col in range(len(grid[row])):
            # Add element to row sum
            row_sum += grid[row][col]
        
        # Print result for this row
        print(f"Row {row}: {row_sum}")
